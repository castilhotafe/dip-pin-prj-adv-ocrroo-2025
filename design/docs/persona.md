# Persona: Jake Finn

## Background
Jake is a 27-year-old university student and junior software developer from Perth, Western Australia. He studies programming and regularly watches coding tutorials to improve his skills. Jake has low vision and uses accessibility features such as screen readers and high-contrast display settings.

## Goals
- Learn programming concepts from coding videos
- Follow code demonstrations more easily
- Extract code examples displayed on screen
- Study independently

## Needs

- Simple and accessible interface
- High contrast display options
- Easy navigation through video content
- Readable extracted code
- Minimal steps to perform OCR extraction
